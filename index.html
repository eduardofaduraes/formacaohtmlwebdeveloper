O que são Clients?

É quem requisita e usufrui dos arquivos estáticos, como: HTML, CSS, JavaScript, Imagens e etc, que estão armazenados e são disponibilizados por um servidor na internet.

Geralmente para se acessar os websites e/ou aplicações web, muitas vezes responsivos, são utilizados browsers (navegadores) ou Aplicativos dedicados.

O que são Servers?

Os Servers são hardwares locais ou remotos (cloud*) que armazenam e disponibilizam diversos tipos de arquivos e softwares.

Eles podem ser:

  • Servidores de Sistemas Operacionais (Windows Server, CentOS, Apache, etc)
  • Servidores de Banco de Dados (Azure SQL Database, Microsoft SQL Server, MySQL, Oracle, etc)
  • Servidores de Monitoramento (ZABBIX e Grafana)
  • Servidores de Dados e Arquivos (SAMBA)
  • Servidores de Segurança (Firewall)
  • Servidores de Streaming
  • Servidores de Websites e Blogs
  • Servidores de E-mail

É quem disponibiliza os arquivos requisitados pelos Clients.

Como é uma estrutura básica de um HTML?

A estrutura básica de um arquivo HTML é composta da seguinte forma:

  • Tag "< !DOCTYPE html >": Declaração da definição do tipo de arquivo que será interpretado pelo navegador.
  • Tag "< html >": Início da estrutura do Website.
  • Tag "< head >": "Cabeça" do website, onde são declaradas as integrações entre arquivos, linguagem do website, título da página e afins.
  • Tag "< body >": "Corpo" do website, tudo declarado aqui será exibido no navegador e consequentemente ao usuário.
  • Tag "< p >": Cria parágrafos dentro da estrutura da página.
  • Tag "< a >": Cria links dentro de uma ou mais palavras, podendo direcionar para uma certa página web externa ou dentro da mesma página.
  • Tag "< i >": Deixa um determinado texto em itálico.
  • Tag "< u >": Deixa um determinado texto sublinhado.
  • Tag "< strong >": Deixa um determinado texto em negrito.
  • Tag "< ol >": Cria uma lista ordenada.
  • Tag "< ul >": Cria uma lista não ordenada.
  • Tag "< li >": Cria elementos da lista, ordenada ou não.

Entendendo a diferença entre listas ordenadas e não ordenadas

A Lista Ordenada, como o próprio nome já diz, é uma lista que segue uma ordem.

Por exemplo:

  1. Início
  2. Desenvolvimento
  3. Conclusão

E uma Lista Não-ordenada segue o mesmo conceito, colocando "bolinhas" ao invés de números:

  • Início
  • Desenvolvimento
  • Conclusão
